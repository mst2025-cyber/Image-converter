# 画像コンバーター (JPG / WebP)

複数の品質・圧縮パターンで画像を JPG または WebP に一括変換する Web アプリケーション。

## 特徴

- **ブラウザのみで完結** — サーバー不要
- **JPG / WebP 出力** — バッチ全体で出力形式を選択
- **複数品質で同時変換** — 25%, 50%, 75%, 100% の 4 段階プリセット（カスタマイズ可）
- **変換前後を比較** — リアルタイムプレビュー、ファイルサイズ、圧縮率
- **一括ダウンロード** — 複数ファイルを ZIP で出力
- **右クリック保存** — プレビューから個別ファイルを `{元名}_{品質}pct.jpg` / `.webp` 形式で保存

## 使用方法

### 1. 出力形式を選択
左側「変換設定」で **JPG** または **WebP** を選択します。アップロードしたすべての画像が同じ形式で変換されます。

### 2. 画像アップロード
対応形式（PNG / JPEG / WebP / GIF / AVIF / BMP）をドラッグ&ドロップするか、「ファイルを選択」をクリックします。複数ファイルにも対応しています。

### 3. プリセット選択
「実行プリセット」から、変換に使用するプリセットをチェックします。

### 4. プレビュー確認
右側で元画像と変換済み画像を比較し、ファイルサイズと圧縮率を確認します。

### 5. ダウンロード
- **ZIP 一括** … 「すべてをZIPダウンロード」ボタン
- **個別保存** … プレビューサムネイルを右クリック →「名前を付けて保存」

## 使い分けガイド

### 画像タイプ別の目安

| 画像タイプ | 向く出力 | 品質の目安 | 補足 |
|-----------|---------|-----------|------|
| 写真 | JPG / WebP | 75〜85% | サイズと画質のバランスが取りやすい |
| イラスト | WebP / JPG | 85〜95% | 境界・グラデーションは高め推奨 |
| 図形・表 | WebP / JPG | 90〜100% | 線や文字のにじみを抑える。元が小さい PNG なら変換不要なことも |
| スクリーンショット | WebP 推奨 | 90〜100% | 文字の読みやすさ優先 |

### 品質と圧縮率について

**品質スライダーと圧縮率は比例しません。**

- **品質** … 変換時にどれだけ細部を捨てるかの設定
- **圧縮率** … 元ファイルサイズに対して実際にどれだけ小さくなったか（結果テーブルに表示）

画像の中身によって結びつき方が変わります。

- **図形・表** … 高品質でも圧縮率が高い（よく小さくなる）ことが多い
- **すでに小さい PNG** … 圧縮率 0% やマイナスもあり得る
- **写真** … 品質を下げるほどファイルは小さくなりやすいが、完全な直線ではない
- **JPEG の再変換** … 圧縮率はあまり動かないのに、画質だけ劣化することがある

### 不可逆変換の注意

- JPG と WebP（本アプリ）はいずれも **不可逆（ロッシー）圧縮** です
- 同じ画像を繰り返し変換するたびに **画質が劣化** します（品質 100% でも可逆ではありません）
- 何回変換されたかの履歴はファイルに残りません
- 元が JPEG の画像を再度 JPG に変換する用途には向きません

### WebP について

本アプリの WebP 出力はブラウザの Canvas API を利用した **不可逆 WebP のみ** です。可逆（ロッスレス）WebP には対応していません。

## プリセット管理

### デフォルトプリセット

| プリセット | 品質 | 用途 |
|----------|------|------|
| 25% - 最軽量 | 25% | 最小ファイルサイズ |
| 50% - 軽量 | 50% | 軽量な圧縮 |
| 75% - 標準 | 75% | 標準品質 |
| 100% - 高品質 | 100% | 最高品質 |

プリセット名・品質値は編集可能で、LocalStorage に自動保存されます。

## 出力ファイル名

```
{元ファイル名(拡張子除く)}_{品質}pct.jpg
{元ファイル名(拡張子除く)}_{品質}pct.webp
```

同名衝突時は `_2`, `_3` … が付きます。

## ブラウザ対応

- Chrome (最新版)
- Firefox (最新版)
- Safari (最新版)
- Edge (最新版)

WebP 出力はブラウザのエンコード対応が必要です。非対応ブラウザでは WebP 選択肢が無効になります。

## 技術仕様

### 使用技術

- **Canvas API** — 画像 → JPG / WebP 変換
- **FileReader API** — ファイル読み込み
- **Blob API** — ファイルダウンロード
- **JSZip.js** — ZIP ファイル生成
- **LocalStorage** — プリセット・出力形式の保存

### ファイル構成

```
product/
├── index.html       # UI 構造
├── styles.css       # スタイル
├── app.js           # ロジック
└── README.md        # このファイル
```

## 注意事項

- 処理はブラウザのメモリ上で実行されるため、大容量画像は時間がかかる場合があります
- ページを離れるとメモリが解放されます
- GIF は先頭フレームのみ変換されます
